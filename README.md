# app-dashboard
Dashboard Web desenvolvido para visualizar e gerenciar dados de forma dinâmica, com integração direta ao banco de dados. A aplicação utiliza AJAX para comunicação assíncrona e jQuery para manipulação do DOM, garantindo uma experiência fluida, sem recarregar a página. O foco do projeto é praticar integração front-end com banco de dados e consumo de dados em tempo real.
